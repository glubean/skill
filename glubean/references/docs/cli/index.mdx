# CLI

The glubean CLI runs API verifications from the terminal. Use it for local development, CI pipelines, and team workflows.

## Install

```bash
# Global install
npm install -g @glubean/cli

# Or run without installing
npx glubean@latest <command>

# Or add to your project
npm install -D @glubean/cli
```

## Quick start

All commands work with `npx glubean@latest` — no global install required.

```bash
# Scaffold a new project
npx glubean@latest init

# Run all tests in tests/
npx glubean@latest run tests/

# Run a single file
npx glubean@latest run tests/users.test.ts
```

If you installed the CLI globally (`npm install -g @glubean/cli`), you can drop the `npx glubean@latest` prefix and use `glubean` directly.

## Get help

```bash
npx glubean@latest -h              # All commands
npx glubean@latest run -h          # Options for a specific command
```
