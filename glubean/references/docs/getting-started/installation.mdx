# Install & Verify

Use this page when you want a working Node-based Glubean setup in a few minutes.

## What you need

- Node.js 20+

## 1. Install the extension (recommended)

The extension is not required to start — you can write and run tests with just the CLI. But for debugging failures, inspecting traces and response schemas, and iterating on tests, the extension is the recommended default.

Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=Glubean.glubean) or the [VSIX releases](https://github.com/glubean/vscode/releases). Works in VS Code, Cursor, and other compatible editors.

## 2. Initialize a project

Project init still uses the CLI, but you do not need a global install. Use `npx`:

```bash
npx glubean@latest init
```

This scaffolds a Node project with `package.json`, test scripts, `tests/`, `explore/`, and env files.

## 3. Verify the CLI entry

If you want to verify the CLI explicitly:

```bash
npx glubean@latest --help
```

## Optional: install the CLI globally

If your team prefers a global binary:

```bash
npm install -g @glubean/cli
```

That gives you the `glubean` command on PATH (plus `gb` as a shorthand).
